CSS Selectors

Applying CSS to Different Parts of HTML

1. The element selector targets elements based on their HTML tag name.

  1. Class Selectors target elements based on the value of the class attribute.
  2. ID selectors target elements based on the value of the id.
  3. Attribute Selectors target elelments based ont their attributes and values.
  4. The unoiversal selector selector targets all elements.
